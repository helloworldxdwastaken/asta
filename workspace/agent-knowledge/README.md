# Agent Knowledge

Put per-agent documents here. Asta retrieves relevant snippets when you route
messages with `@Agent Name: ...`.

Structure:

- `<agent-id>/sources/` - raw research and collected material
- `<agent-id>/references/` - stable frameworks/playbooks
- `<agent-id>/notes/` - quick internal notes and assumptions

Use text-friendly formats (`.md`, `.txt`, `.json`, `.yaml`, `.csv`, `.html`).
